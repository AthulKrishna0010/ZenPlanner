# 📝 Mentor-Student Assignment Tracker

A full-stack web application to help **mentors** assign and track tasks, and for **students** to view, manage, and submit assignments. Built with **React.js** (frontend), **Spring Boot** (backend), and **MongoDB** (database).

## 🌟 Features

### 👨‍🏫 Mentor Side
- Create, update, and delete assignments for individual students.
- View assignments by student.
- Track assignment statuses and deadlines.
- View uploaded student submissions.

### 👩‍🎓 Student Side
- View personal assignments.
- Update status (e.g., In Progress, Completed).
- Upload files for submitted assignments.
